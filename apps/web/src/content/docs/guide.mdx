# AI-Guide

Der Chat unten rechts hilft dir bei Fragen rund um powere.ch (Daten, Loader, Methodik).  
**Wichtig:** Antworten basieren ausschließlich auf internem Kontext.  
Wenn die Info nicht im Kontext ist, sagt der Chat das ehrlich.

## Nutzung
- **Frage tippen** → Enter sendet, **Shift+Enter** für Zeilenumbruch  
- Antworten erscheinen **streaming** (Zeichen-für-Zeichen)  
- Quellen siehst du unter der Antwort

> Für Entwickler*innen gibt es eine ausführliche Analyse-Ansicht:  
> **“AI-Guide Studio”** (Streaming, Token-Timings, Retrieval-Ranking, Debug).
